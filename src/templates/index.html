Introduction

This web-site gives an access to API for generating customized passwords.

How to use

    The step-by-step guide for using API:
  1. Create HTTP-request with GET method using this template:
    http://website_name/generate?Parameter1&Parameter2
  2. Recieve JSON-response:
    • If request was successful the status-code of response would be 200 and response would contain only ‘string’ attribute with generated password.
    • If you gave wrong parameters the response will be with statuts 400 and contain only ‘error’ attribute with error clarification.
  3. Extract generated password from JSON-response and use it where you need.

    Parameters:
  • count - defines the length of password, must be an integer number, this parameter is not optional;
  • letters - adds upper-case and lower-case letters, must be 'on' or include custom letters;
  • upper-case - adds upper-case letters (A-Z), must be 'on';
  • lower-case - add lower-case letters (a-z), must be 'on';
  • digits - adds digits (0-9), must be 'on' or include custom digits;
  • special_symbols - adds special symbols (punctuation and others), must be 'on' or include punctuation symbols.
